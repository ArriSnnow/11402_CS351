# 11402_CS351

## AI-assisted Software Development

AI-assisted Software Development focuses on using modern AI tools to plan, implement, test, debug, and document software more effectively. The course covers:

- Prompt engineering
- AI-augmented coding workflows
- Code review
- Refactoring
- Responsible use of AI in real-world development

The course emphasizes practical projects and best practices for integrating AI into the software lifecycle.
